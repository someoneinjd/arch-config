# Neovim Config
This is a minimal (neo)vim configuration without external dependencies

## Key Bindings
| Key         | Action                       |
| ----------- | ---------------------------- |
| Meta + e    | Open file explorer           |
| Meta + t    | Open terminal                |
| Meta + n    | Goto next tab                |
| Meta + p    | Goto previous tab            |
| Meta + d    | Close current tab            |
